// Sobre el proyecto

Obtuvimos una plantilla en HTML5up llamada Phantom, y modificamos sus archivos HTML para hacer la web.

Junto a la plantilla que obtuvimos en HTML5up venia un código CSS, predeterminado para esa plantilla, lo que hicimos fue modificar o crear algún dato en el código, para elegir el color de los recuadros o para crear un estilo para el texto extensible.
