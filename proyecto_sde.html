// Sobre el proyecto

Control de acceso físico montado y programado íntegramente desde cero con Arduino. El sistema integra LEDs de estado, un teclado on/off y un servomotor para la apertura automática de la puerta cuando la contraseña es correcta.

El proyecto se desarrolló en dos fases: primero una simulación digital para validar la lógica del circuito, y después el montaje físico real con todos los componentes soldados y cableados a mano.
