2024 · Controller hardware + Unity game
Mando + Videojuego de Carreras
ArduinoUnityElectrónicaC#

Mando físico construido desde cero con Arduino Leonardo y una placa de topo con circuito soldado manualmente. Totalmente funcional y conectado a un videojuego de carreras desarrollado en Unity.

// Detalles del proyecto
01 Base construida con placa de topo y soldadura manual de componentes electrónicos.
02 Arduino Leonardo como microcontrolador principal para lectura de inputs en tiempo real.
03 Videojuego de carreras creado en Unity con integración del mando como controlador HID.
04 Programación de la comunicación serial entre Arduino y Unity.
// Tecnologías
Arduino Leonardo
Unity
C#
Soldadura electrónica
Comunicación HID
