El reto: construir un acceso seguro desde cero

El objetivo principal de este proyecto fue diseñar e implementar un sistema de control de acceso funcional utilizando únicamente componentes electrónicos básicos y una placa Arduino UNO. El sistema permite o deniega la entrada según la autenticación del usuario mediante tarjeta RFID.

El proceso comenzó con el diseño del esquema de conexiones en papel, seguido del montaje físico en protoboard. La programación se desarrolló de forma incremental: primero la lectura del módulo RFID, luego la lógica de comparación de UIDs, y finalmente la integración con el servomotor y el display LCD para dar feedback visual al usuario.

Uno de los mayores aprendizajes fue la gestión de la comunicación SPI entre el Arduino y el lector RFID, así como la calibración del servomotor para garantizar un ángulo de apertura preciso. El proyecto finalizó con éxito con una demo en clase donde el sistema reconocía tarjetas autorizadas y bloqueaba intentos con tarjetas no registradas.
